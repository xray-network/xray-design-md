Prompt Example

Create a send form UI as a single, self-contained HTML file with all CSS embedded in a <style> element. Use only HTML and CSS—no frameworks, build tools, external UI libraries, external stylesheets, or additional files. Follow DESIGN.md strictly as the design source of truth: https://design-md.xray.app/DESIGN.md. Required output: send-form.html only. Layout constraints: responsive on desktop and mobile, max-width 800px, centered content. Theme requirements: include a working CSS-only dark/light theme toggle with #000 background in dark mode and #fff background in light mode, and ensure readable contrast in both themes. UI content: include a security icon and concise trust/security messaging. Flow requirements: represent a two-step crypto multi-asset send flow where step 1 is a form to choose multiple assets, enter amounts, and set a recipient; step 2 is an awaiting-signature confirmation/loading state. Use semantic HTML, keyboard-accessible controls, clear form labels, and polished spacing and typography consistent with DESIGN.md. Return only the complete contents of send-form.html.
